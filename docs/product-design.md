# 一、产品概述

## 1.1 项目背景

DATI（Digital AI Personality Test）数智人格测试是一款面向中国互联网用户的性格测试产品，主题聚焦于**AI时代的生存状态与态度**。对标产品为SBTI（性格色彩测试），但更具时代共鸣感和传播属性。

## 1.2 产品定位

| 项目 | 说明 |
|------|------|
| 产品类型 | 轻量级人格测试/社交传播类H5 |
| 目标用户 | 18-35岁年轻人、互联网从业者、打工人 |
| 核心价值 | 30道题，3分钟，揭示用户在AI时代的数智人格 |
| 传播场景 | 朋友圈分享、微博话题、微信群传播 |

## 1.3 核心功能

| 功能模块 | 说明 |
|---------|------|
| 首页 | 主题展示、开始测试入口 |
| 答题模块 | 30道题目，单选，实时进度 |
| 过渡动画 | AI分析loading效果 |
| 结果展示 | 数智人格类型、雷达图、特质描述 |
| 分享功能 | 生成海报图，支持朋友圈分享 |

## 1.4 人格体系

- **3大核心维度** × **3种状态** = **23种标准人格**
- **4种隐藏人格**（神仙、山顶洞人、矛盾体、资本家）
- **总计27种人格类型**

# 二、3大核心维度

## 2.1 维度定义

| 维度 | 🔴 负向极端 | 🟡 中立型 | 🟢 正向极端 |
|------|-----------|---------|-----------|
| AI心态指数 | AI恐慌者 (-20~-5) | 佛系随缘人 (-5~+5) | AI躺平党 (+5~+20) |
| AI融合指数 | AI绝缘体 (-20~-5) | 工具使用者 (-5~+5) | AI共生者 (+5~+20) |
| 数据主权指数 | 数据守财奴 (-20~-5) | 理性交易者 (-5~+5) | 数据无所谓者 (+5~+20) |

## 2.2 计分规则

- 每个维度 **10道题**，共30道题
- 每道题 **5个选项**：-2、-1、0、1、2
- 每维度得分范围：**-20 ~ +20**
- 得分区间判定：
  - 5 ~ 20：正向极端
  - -5 ~ 5：中立型
  - -20 ~ -5：负向极端

## 2.3 人格计算公式

```
3个维度 × 3种状态 = 27种人格类型
```

## 7.4 核心算法

### 人格判定算法

```typescript
function calculatePersonality(answers: number[], questions: Question[]) {
  // 1. 计算各维度总分
  const scores = { anxiety: 0, neutral: 0, laidback: 0 };
  
  answers.forEach((answerIndex, questionIndex) => {
    const question = questions[questionIndex];
    const selectedOption = question.options[answerIndex];
    
    scores.anxiety += selectedOption.scores.anxiety;
    scores.neutral += selectedOption.scores.neutral;
    scores.laidback += selectedOption.scores.laidback;
  });

  // 2. 确定基础人格
  const maxScore = Math.max(scores.anxiety, scores.neutral, scores.laidback);
  let baseType = 'neutral';
  if (scores.anxiety === maxScore) baseType = 'anxiety';
  else if (scores.laidback === maxScore) baseType = 'laidback';

  // 3. 检测隐藏人格
  const hiddenType = detectHiddenType(scores, answers, questions);

  return { baseType, hiddenType, scores };
}
```
