# DATI · 数智人格测试

> AI时代人格探测仪 — 测测你在AI时代是什么牛马

## 产品介绍

DATI（Data & AI Trait Index）是一款面向AI时代的人格测试产品，通过三个核心维度探测用户与AI的关系类型：

- **AI心态指数** — 对AI发展的焦虑/期待程度
- **AI融合指数** — 与AI的融合/依赖程度
- **数据主权指数** — 数据隐私/控制意识

## 27种人格类型

测试通过30道题目，输出27种不同的AI时代人格类型，每种人格有专属颜色、特质描述和建议。

## 技术栈

- 纯前端：HTML5 + CSS3 + JavaScript
- 无框架依赖
- Canvas绘制雷达图和分享图
